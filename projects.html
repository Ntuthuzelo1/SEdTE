About Our Projects

In this, the aim is to give you a glimps or a summary and keep track of the projects the company is currently working on and or has worked on.

UMZI wakwaXHOSA

With isiXhosa as our initial initiative, this project has been designated with the purpose of enhancing and digitalizing our native tongues.It aims to offer a useful application as a substitute for traditional educational methods for teaching isiXhosa. It aims to include many dialects of isiXhosa, including Amaqhalo, Izaci, Iirayi-rayi, iintsomi, imibongo, and amabali.

Masifundeni

This project seeks to build an application which aims at bringing new ways of accessing xhosa novels, with the limited number of programs which accomodates our languages, this provides a flexible platform for users to install the application in their smartphones and be able to read anywhere. Not only does this application seeks to target isiXhosa, in the long term run it seeks to accomodate other languages.
